# 智慧相框

## 動機

假如晚上已經關燈上床準備睡了，突然想到明天有一個新行程需要紀錄，但懶得拿出手機，想要以更便利的方式記錄下來。

## 功能

上傳圖片到雲端由雲端抓取資料夾內圖片並在顯示器上輪播，按下enter後可以口述之後行程，會用程式紀錄於google calendar，且在前10分鐘語音提醒行程。
